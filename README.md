# Retail Analytics Dashboard

An advanced e-commerce analytics dashboard built with Python, Streamlit, Pandas, and Plotly.

## Features

- KPI metrics: Total Revenue, Orders, Customers, etc.
- Interactive filters for Country, Year, Month, Product, Customer
- Advanced visualizations: Sales trends, top products, country analysis, etc.
- RFM customer segmentation
- Interactive data table with download option

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Place `Online Retail.xlsx` in the project directory
3. Run: `streamlit run app.py`

## Dataset

The dataset `Online Retail.xlsx` contains transaction data with columns: InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country.

## Deployment

Deployable on Streamlit Cloud with code hosted on GitHub.
